# AI-Powered Phishing Email Detection System

## Project Overview

Phishing attacks represent one of the most significant cybersecurity threats, where malicious actors attempt to deceive users into revealing sensitive information such as credentials, financial data, or personal information.

This project presents an AI-based system designed to detect phishing emails using Machine Learning and Natural Language Processing (NLP) techniques. The system classifies emails as either **Phishing** or **Legitimate (Ham)** and provides interpretable outputs, including confidence scores and explanatory indicators.

Additionally, the system incorporates an attack simulation module to evaluate robustness against realistic phishing scenarios.

---

## Objectives

- Develop a machine learning-based phishing detection system
- Apply NLP techniques for email text preprocessing and feature extraction
- Improve interpretability through explainability methods
- Simulate phishing attacks to evaluate system performance
- Provide a lightweight and extensible architecture for cybersecurity applications

---

## Key Features

### Phishing Email Classification
- Binary classification of emails (Phishing / Legitimate)
- Supervised learning using labeled datasets

### Natural Language Processing Pipeline
- Text cleaning and preprocessing
- Tokenization and normalization
- Feature extraction using TF-IDF

### Confidence Score Generation
- Outputs probability scores associated with predictions
- Provides insight into model certainty

### Explainability Module
- Identifies suspicious words and patterns
- Highlights indicators such as urgency, impersonation, and suspicious links
- Provides reasoning for model predictions

### Attack Simulation Module
- Generates synthetic phishing emails
- Enables testing under controlled adversarial scenarios
